# Ruby on Rails Api

This application are a API-only Applications with PostgreSQL

## Prerequisites

* Ruby		2.7.0
* Rails 	13.0.6

## Install & Run project

```bash
$ bundle install
$ rails db:create
$ rails db:migrate
$ rails db:seed
$ rails server
```
